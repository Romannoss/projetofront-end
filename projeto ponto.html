O PowerCenter é um aplicativo de ETL - Extract,Transform and Load. É uma ferramentas de software cuja função é a extração de dados de diversos sistemas, transformação desses dados conforme regras de negócios e por fim o carregamento dos dados, geralmente, para um Data Mart e/ou Data Warehouse, porém nada impede que também seja para enviar os dados a um determinado sistema da organização ou geração de relatório.
